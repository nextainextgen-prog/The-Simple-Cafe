> เช็กลิสต์สำหรับส่งให้ผู้จ้าง — ติ๊กทีละข้อ ยิ่งได้ครบ เว็บยิ่งเสร็จเร็วและตรงจริง
> ของที่ยังไม่มา ผมใส่ placeholder ไปก่อนได้ ไม่บล็อกงาน
> อัปเดต: 2 ก.ค. 2026

## กลุ่ม A — แบรนด์ (ขอก่อนสุด เพราะกระทบทั้งเว็บ)
- [ ] โลโก้ไฟล์ต้นฉบับ (ไฟล์ .ai/.svg/.png พื้นใส ทั้งแบบเต็มและแบบย่อ/ไอคอน)
- [ ] สีแบรนด์ (ถ้ามี brand guide) — ถ้าไม่มี ผมเสนอโทนให้เลือก
- [ ] ฟอนต์ที่อยากใช้ (ถ้ามี) — ถ้าไม่มี ผมเลือกฟอนต์ไทยที่อ่านง่าย+ดูพรีเมียมให้
- [ ] สโลแกน/tagline สั้นๆ ของแบรนด์ (ถ้ามี)
- [ ] ความสัมพันธ์กับ "MADE CAKE" — จะแสดงบนเว็บไหม (โลโก้/ชื่อ/เครดิต) หรือไม่แสดง

## กลุ่ม B — เนื้อหาแต่ละหน้า

### หน้าแรก
- [ ] ประโยค Hero (พาดหัวหลัก 1 ประโยค + รองอีก 1 บรรทัด) — หรือให้ผมร่างให้เลือก
- [ ] จุดเด่น 4-6 ข้อ (Why Simple Cafe) — อบสดใหม่ / ส่งทั่วประเทศ / 6+ ปี / รับผลิต OEM ทุกรูปแบบ ฯลฯ

### About / เกี่ยวกับเรา
- [ ] เรื่องราวแบรนด์ (Brand Story) — เริ่มต้นยังไง ทำไมทำเบเกอรี่ (2-3 ย่อหน้า หรือเล่าให้ผมเรียบเรียง)
- [ ] กระบวนการผลิต (อบสด ไม่ใส่สารเติมแต่ง ฯลฯ) อธิบายเป็นสเต็ปได้ยิ่งดี
- [ ] มาตรฐานการผลิต / ใบรับรอง (อย. / GMP / HACCP ถ้ามี — เอาโลโก้/เลขที่ด้วย)

### สินค้า (Products)
- [ ] รายการสินค้าจริง แยกหมวด (เค้ก / คัพเค้ก / ขนมปัง / ครัวซองต์ / ซาวโดว์ / อื่นๆ)
- [ ] แต่ละตัว: ชื่อสินค้า + คำอธิบายสั้น + ราคาปลีก (ถ้าจะโชว์) + รูป
- [ ] ระบุด้วยว่าตัวไหน Best Seller / New เพื่อเอาขึ้นหน้าแรก
- [ ] เกรดสินค้า — อยากแยกโชว์ "มาตรฐาน vs พรีเมียม" ไหม (เนยฝรั่งเศส/แป้งญี่ปุ่น ฯลฯ)

### จัดเบรก (Catering)
- [ ] แพ็กเกจจัดเบรก + ราคา/ช่วงราคา (หน้านี้โชว์ราคาได้ ช่วยลูกค้าประเมินงบ)
- [ ] เมนูในแต่ละชุด (เบเกอรี่ + เครื่องดื่ม + ของว่าง)
- [ ] เงื่อนไข: ขั้นต่ำกี่ที่ / จองล่วงหน้ากี่วัน / พื้นที่ส่ง
- [ ] รูปงานจริง (ชุดเบรก / table setting / ออกบูธอีเวนต์)

### B2B / Wholesale
- [ ] เงื่อนไขลูกค้าส่ง: ยอดสั่งขั้นต่ำ (MOQ) / รอบส่ง / เงื่อนไขราคาส่ง
- [ ] กลุ่มลูกค้าที่รับ (คาเฟ่ / ร้านอาหาร / โรงแรม / จัดเลี้ยง)
- [ ] รับผลิต OEM แบบไหนได้บ้าง (ทำแบรนด์ลูกค้า / คิดสูตรเฉพาะ ฯลฯ)
- [ ] หมายเหตุ: หน้านี้ **ไม่โชว์ราคาส่ง** ใช้ปุ่ม "ขอใบเสนอราคา" แทน (ตาม research)

## กลุ่ม C — Social Proof (ช่วยปิดการขายมาก)
- [ ] โลโก้ลูกค้าองค์กร / ร้านที่ส่งประจำ (ขออนุญาตใช้โลโก้ด้วย)
- [ ] รีวิว/คำชมจากลูกค้า (ข้อความ + ชื่อร้าน/ชื่อคน ถ้าให้ลงได้)
- [ ] ตัวเลขที่โชว์ได้ เช่น "ลูกค้า 60+ ราย / ประสบการณ์ 6+ ปี / ส่งกี่จังหวัด"

## กลุ่ม D — รูปภาพ (หัวใจของเว็บ เน้นโชว์ของ)
- [ ] รูปสินค้ารายตัว คุณภาพสูง แสงสวย (ยิ่งเยอะยิ่งดี — นี่คือพระเอกของเว็บ)
- [ ] รูปหน้าร้าน / คาเฟ่ / บรรยากาศ
- [ ] รูปเบื้องหลังการอบ / ทีมงาน (เอาไว้หน้า About สื่องานคราฟต์)
- [ ] รูปงานจัดเบรก / อีเวนต์จริง
> ถ้ายังไม่มีรูปโปร ผมใส่ placeholder ไปก่อน แล้วค่อยเปลี่ยนทีหลังได้ แต่แนะนำให้ลงทุนถ่ายรูปโปร คุ้มสุด

## กลุ่ม E — ติดต่อ + SEO ท้องถิ่น
- [ ] ชื่อร้านเต็ม (ไทย + อังกฤษ)
- [ ] ที่อยู่จริง (สำหรับแผนที่ + Google Business Profile)
- [ ] เบอร์โทร
- [ ] LINE OA (ID / ลิงก์) — จะเป็นปุ่มเด่นทุกหน้า
- [ ] Facebook / Instagram / TikTok (ถ้ามี)
- [ ] อีเมลติดต่อ
- [ ] เวลาทำการ
- [ ] พื้นที่ให้บริการส่ง (ขอนแก่นส่งทุกวัน + ทั่วประเทศส่งยังไง)
- [ ] มี Google Business Profile อยู่แล้วไหม (ถ้ามี ขอสิทธิ์เข้าจัดการ)
- [ ] โดเมนเว็บ — มีอยู่แล้วหรือต้องจดใหม่ (เช่น simplecafe.com / .co.th)

## กลุ่ม F — ฟอร์ม (ปลายทางข้อมูล)
- [ ] อยากให้ข้อความจากฟอร์มส่งไปที่ไหน (อีเมล / LINE / Google Sheet)
- [ ] ใครเป็นคนรับเรื่อง/ตอบลูกค้า

## หมายเหตุจากผม (นาย)
- ของที่ **จำเป็นต้องได้ก่อนเริ่มดีไซน์จริง**: โลโก้ + สีแบรนด์ + รูปสินค้าตัวอย่าง 5-10 รูป + ข้อมูลติดต่อ
- ที่เหลือทยอยส่งได้ ผมทำโครง + ใส่ placeholder ไปก่อนได้เลย ไม่ต้องรอครบ
- ถ้าเจ้าของไม่มีเวลาเขียนเนื้อหา (Brand Story / จุดเด่น) เล่าให้ผมฟังคร่าวๆ ผมเรียบเรียงให้ได้
