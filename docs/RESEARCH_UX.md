## 2. เทรนด์ดีไซน์เว็บอาหาร/เบเกอรี่ 2025-2026 (ยืนยัน 3-0)

- **Bold minimalism** = เลย์เอาต์สะอาด + สีเน้นจำกัด + micro-interactions (แต่แนว fine-dining/พรีเมียมใช้ accent เดียวแบบ restrained ไม่ vibrant จัด)
- **ภาพ:** hero ภาพสินค้าเต็มจอ (full-bleed) คุณภาพสูงราวแกลเลอรี + ภาพกระบวนการอบสื่องานคราฟต์
- **สี:** พื้นกลาง (ขาว/ครีม/off-white) + สีเน้นเดียว (maroon/terracotta/มูทเอิร์ธ/ทองด้าน)
- **เลย์เอาต์:** กริดโปร่ง whitespace กว้าง + card-based product display
- **Motion:** micro-interaction / scroll animation แบบ subtle (อย่าเยอะจนเสียความพรีเมียม)
- **Mobile-first:** ทราฟฟิกเว็บร้านอาหาร >60% มาจากมือถือ → sticky nav + ปุ่ม LINE/โทร/ขอราคาแตะง่าย
- **ข้อที่โดนตีตก:** "food photography เป็นเครื่องมือปิดการขายที่ได้ผลสุด" — verify แล้ว 0-3 (over-claim) ภาพสำคัญจริงแต่ไม่ใช่ปัจจัยเดียวที่ปิดการขาย

## 3. โครงสร้างเว็บ (Sitemap) — ยืนยัน 3-0

หน้าที่ควรมี:
- **Home** — hero ภาพ + จุดขาย + แยกทางเข้า retail/B2B/catering
- **About** — เรื่องราว + กระบวนการอบ (สื่อคราฟต์ + ประสบการณ์ 6 ปี)
- **สินค้า** — แยกหมวด (เค้ก/ขนมปัง/ครัวซองต์/ซาวโดว์) + แยกเกรดมาตรฐาน/พรีเมียม
- **Wholesale / B2B** — แคตตาล็อก + ขอใบเสนอราคา + Who We Serve + แผนที่รัศมีส่ง + MOQ (ขั้นต่ำ)
- **Catering / จัดเบรก** — เมนูเป็นหมวด + ช่วงราคา/แพ็กเกจ + CTA หลายทาง
- **จัดส่ง** — แยกขอนแก่น (ทุกวัน) vs ทั่วประเทศ
- **Contact** — ข้อมูลติดต่อ + แผนที่ + ฟอร์ม + LINE

Nav: แยกสายธุรกิจให้ชัด (ปลีก / B2B / จัดเบรก) แบบ Pophams

## 5. Conversion — เปลี่ยนคนเข้าเว็บเป็นลูกค้า (ยืนยัน 3-0)

- **หน้า B2B ไม่โชว์ราคาสาธารณะ** → ใช้ปุ่ม "ขอใบเสนอราคา / สมัครลูกค้าส่ง / ขอตัวอย่างสินค้า" นำไปฟอร์ม (เป็น lead-gen ไม่ใช่ e-commerce)
- **หน้า Catering โชว์ราคา/ช่วงราคาได้** → ช่วยลูกค้าประเมินงบ
- **ฟอร์มขอใบเสนอราคา** เก็บฟิลด์เบาๆ:
  ชื่อธุรกิจ / ที่ตั้ง / ประเภทธุรกิจ (dropdown: คาเฟ่-ร้านอาหาร-โรงแรม-จัดเลี้ยง) / ปริมาณ+ความถี่ / สินค้าที่สนใจ / ชื่อ-อีเมล-โทร + แจ้ง "ตอบกลับใน 2-3 วันทำการ"
- **Social proof:** section "Who We Serve" + จำนวนลูกค้า/คาเฟ่ที่ส่งจริง + โลโก้พาร์ตเนอร์ (โดยเฉพาะในขอนแก่น) + รีวิว
- **LINE OA:** ปุ่ม LINE เด่นทุกหน้า ใช้รับออเดอร์ + ขอใบเสนอราคา (เป็นช่องแชตช้อปหลักของไทย — confidence: medium, แหล่ง primary เป็นเว็บ LINE เอง)
  - https://lineforbusiness.com/th
- แหล่ง B2B conversion: https://www.addtoquote.com/blog/request-a-quote-template-examples-for-b2b

## 6. การโชว์สินค้า (Product Catalog) — ยืนยัน 3-0

- จัดหมวดชัด + card-based + whitespace
- catering: จัดเมนูเป็นหมวด (เช้า/กลางวัน/ของว่างบ่าย/เค้ก) แทนลิสต์ก้อนเดียว + CTA 3 ทาง (PDF/ฟอร์ม/สั่ง) แบบ Flour
- B2B: ไม่โชว์ราคา / Catering: โชว์ราคาได้

> ข้อจำกัด: ยังไม่มีตัวอย่างจริงที่โชว์ "หลายเกรด (มาตรฐาน vs พรีเมียม) ในแบรนด์เดียว" — ต้องหา reference เพิ่ม

## สรุปข้อเสนอแนะสำหรับ Simple Cafe

**หน้าเว็บที่ควรมี:** Home / About / สินค้า (แยกหมวด+เกรด) / Wholesale-B2B (+ขอใบเสนอราคา) / Catering-จัดเบรก / จัดส่ง (ขอนแก่น+ทั่วประเทศ) / Contact (+แผนที่+LINE)

**โทนดีไซน์:** มินิมอลพรีเมียม — พื้นครีม/ขาว + สีเน้นเดียวโทนเอิร์ธเข้ม/ทองด้าน, whitespace กว้าง, card-based, ภาพสินค้าเต็มจอเป็นพระเอก, micro-interaction แบบ subtle, mobile-first

**3 อันดับต้องทำก่อน:**
1. ภาพถ่ายสินค้าระดับพรีเมียม + ระบบดีไซน์มินิมอลสีเน้นเดียว (แกนของทั้งเว็บ)
2. หน้า Wholesale/B2B + ฟอร์มขอใบเสนอราคา + social proof (Who We Serve + โลโก้ลูกค้า)
3. SEO ท้องถิ่นขอนแก่น + Google Business Profile + ผูก LINE OA

**สิ่งที่ยังต้องค้นเพิ่ม (open questions):**
- เว็บคาเฟ่/เบเกอรี่ไทย 5 เว็บ พร้อมวิเคราะห์
- คีย์เวิร์ดขอนแก่น + ปริมาณค้นหาจริง
- ตัวอย่างการโชว์หลายเกรดในแบรนด์เดียว
