# ฟอนต์แบรนด์ — วางไฟล์ตรงนี้

เว็บตั้งค่าให้ใช้ฟอนต์ 4 ตัวตามแบรนด์ไกด์ The Simple:

| ฟอนต์ | ใช้กับ | ไฟล์ที่ต้องวาง (ตั้งชื่อตามนี้เป๊ะ) |
|---|---|---|
| **Olimpico** | หัวข้อหลัก (display) | `Olimpico.woff2` |
| **Arrière Garde** | accent / ราคา / badge | `ArriereGarde.woff2` |
| **Longhand LP Bold** | ลายมือ / tagline | `LonghandLP-Bold.woff2` |
| **Noto Sans Thai** | ภาษาไทย (body) | โหลดอัตโนมัติจาก Google Fonts แล้ว — ไม่ต้องทำอะไร |

## วิธีใช้
1. เอาไฟล์ฟอนต์ 3 ตัวแรก (แปลงเป็น `.woff2` ก่อนถ้าเป็น .otf/.ttf) มาวางในโฟลเดอร์นี้
2. ตั้งชื่อไฟล์ให้ตรงตารางด้านบน
3. รีเฟรชเว็บ — ฟอนต์จะขึ้นเองทันที (ระบบตั้ง `@font-face` ไว้ใน `app/globals.css` แล้ว)

> ระหว่างที่ยังไม่มีไฟล์ เว็บจะ fallback เป็น Noto Sans Thai ให้อัตโนมัติ (ไม่พัง)

## แปลง .otf/.ttf เป็น .woff2
ถ้ามีแค่ .otf/.ttf ใช้เครื่องมือแปลงออนไลน์ (เช่น cloudconvert) หรือบอกผมไว้ เดี๋ยวจัดให้
